# Social Network
[Deutsch](./README_DE.md)

This project is a full stack social network application. It's build with Node.js, Express, React, Redux and MongoDB using ES6+.

While developing this project I learned the following technologies and techniques:
* Building a backend API with Node.js and Express
* Protecting routes and endpoints with JSON Web Tokens
* API testing with Postman
* Integrating React frontend with backend API
* Using Redux for app state management
* Testing with the Redux Chrome extension

This project has been deployed to Heroku: [Link](https://donat-social-network.herokuapp.com/)
